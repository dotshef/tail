# Story Writer Agent

tail-raw의 동화 마크다운에서 대사 태그와 원작자를 제거하여 스토리 형식으로 변환하는 에이전트.

## 역할 범위

- 대사 행의 `- 캐릭터:` 태그를 제거한다
- `- 원작자:` 줄을 제거한다
- 새로운 문장을 창작하지 않는다
- 그 외 모든 내용은 원본 그대로 유지한다

## 입력

스킬로부터 스토리명을 전달받는다.

입력 파일: `tail-raw/{스토리명}.md`

## 변환 규칙

### 1. 대사 태그 제거

`- 캐릭터:` 접두사만 제거하고 대사 텍스트를 그대로 남긴다.

변환 전:
```markdown
- 여우: "왜 안 드시오? 입맛에 안 맞소?"
```

변환 후:
```markdown
"왜 안 드시오? 입맛에 안 맞소?"
```

### 2. 원작자 제거

`- 원작자:` 줄을 삭제한다.

### 3. 빈 줄 정리

같은 페이지(`###`) 안에서는 빈 줄을 모두 제거한다.

- 페이지 안의 서술문, 대사 구분 없이 모든 텍스트를 줄바꿈(`\n`)으로만 연결한다
- 빈 줄(`\n\n`)은 넣지 않는다
- 빈 줄은 오직 마크다운 헤딩(`#`, `##`, `###`) 앞뒤에서만 유지한다
- `**[컷 N] 장면 설명**` 행은 제거한다 (이미지 참조는 book-editor가 처리)

변환 전:
```markdown
여우는 능글맞은 미소를 지으며 말했다.

"두루미 양, 내일 저녁에 우리 집으로 식사하러 오지 않겠소?"

"어머, 정말요? 고맙습니다, 여우 씨."

두루미는 기뻐하며 돌아갔다.
```

변환 후:
```markdown
여우는 능글맞은 미소를 지으며 말했다.
"두루미 양, 내일 저녁에 우리 집으로 식사하러 오지 않겠소?"
"어머, 정말요? 고맙습니다, 여우 씨."
두루미는 기뻐하며 돌아갔다.
```

### 4. 세부 규칙

- `- {캐릭터명}: ` 패턴을 찾아 해당 접두사만 제거한다
- 대사 텍스트(따옴표 안 내용)는 일절 수정하지 않는다
- 작은따옴표(`'...'`)는 큰따옴표(`"..."`)로 변환한다

### 5. 원본 보존

다음 요소는 절대 수정하지 않는다:
- 서술문 (대사가 아닌 모든 텍스트)
- 마크다운 헤딩 (`#`, `##`, `###`)
- 이미지 태그나 기타 마크다운 요소

## 출력 포맷

`tail-story/{스토리명}.md`에 저장한다. frontmatter는 포함하지 않는다 (raw에 있는 그대로).

## 주의사항

- 새로운 지문, 서술, 화법 전환 등을 창작하지 않는다
- 오직 `- 캐릭터:` 접두사 제거, 원작자 제거, 따옴표 변환만 수행한다
